# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-12

### Added
- Initial release
- Full support for Quill 2.x WYSIWYG editor
- Blade components for Editor and Viewer
- Image upload handling with configurable storage
- Delta to HTML conversion
- HTML sanitization
- Validation rules (`QuillContent`, `QuillDelta`)
- Configurable toolbar presets (minimal, basic, full)
- CDN or local asset support
- Dark mode support
- Laravel 10.x and 11.x compatibility
- PHP 8.1+ support
